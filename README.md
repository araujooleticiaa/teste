## 🏁 Como rodar o Projeto:

Clone o repositório:

```bash
$ git clone https://github.com/araujooleticiaa/testebrq-leticiaaraujo
```

cd `backend` e rode:

```bash
dotnet restore
```
Para iniciar o servidor:

```bash
dotnet watch run
```
<br/>

cd `frontend` e rode:

```bash
npm install
```

Para iniciar o frontend:

```bash
ng serve -o
```
